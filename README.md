# PostgreSQL Customer Management System

A professional PostgreSQL Customer Management Database project designed for:

- Fiverr portfolio
- GitHub showcase
- SQL practice
- Database management learning
- Business analytics demonstrations

This project simulates a real-world customer management system used by companies to manage customers, employees, services, orders, payments, and support tickets.

---

# 📌 Features

- Relational database design
- Customer management
- Employee management
- Service tracking
- Order processing
- Payment management
- Customer support ticket system
- Business analytics queries
- SQL views
- PL/pgSQL functions
- Query optimization using indexes
